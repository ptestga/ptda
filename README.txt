#PTDA
 Projecto Temático em Desenvolvimento de Aplicações 2017/18
 
 Reabilitação Cardiovascular
 
 Este software foi desenvolvido com recurso ao IDE IntelliJ IDEA, pelo que deverão ser abertos com recurso ao mesmo de forma a garantir que não ocorrem problemas de compatibilidade.

— —“Bugs” atuais e melhoramentos necessários — —
Segue uma versão compilada do software “PTDA4” em formato jar, no entanto, não foi possível criar uma versão final do “script-ptda”, pelo que será necessário compilar e correr este programa a partir do IDE de forma a que este possa ser executado.

O software “PTDA4” não está completamente finalizado. As funcionalidades principais do programa encontram-se implementadas e funcionais. Apenas algumas das funcionalidades secundárias ainda não foram implementadas.
